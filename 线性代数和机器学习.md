# 线性代数
- 求偏导数fx(x₀, y₀)
	求 ∂f/∂x 时，将 y 看作常量，对 x 求导
	求 ∂f/∂y 时，将 x 看作常量，对 y 求导

- 偏导数，连续 -> 可微
	可微 -> 连续
	可微 -> 偏导数存在

- 方向导数
	定义：函数 f(x,y) 在点 P₀ 沿方向 l 的方向导数：
	∂f/∂l |P₀ = limρ→0 [f(x₀+Δx, y₀+Δy) - f(x₀, y₀)] / ρ
	计算公式：若函数可微，则 ∂f/∂l = (∂f/∂x)cosα + (∂f/∂y)cosβ
	其中 cosα, cosβ 为方向 l 的方向余弦

- 梯度
	定义：向量 (∂f/∂x, ∂f/∂y) 称为函数 f(x,y) 的梯度，记作 gradf 或 ∇f
	gradf = (∂f/∂x, ∂f/∂y) = ∂f/∂x · i + ∂f/∂y · j
	
	**几何意义：**
	梯度方向是函数增长最快的方向
	梯度的模是方向导数的最大值
	梯度垂直于等值线（函数值相等的点连起来形成的线）

- 无条件极值
	必要条件：若 z = f(x, y) 在点 (x₀, y₀) 有极值且偏导数存在，则fx(x₀, y₀) = 0, fy(x₀, y₀) = 0
	
	充分条件：设 A = fxx（二阶偏导数），B = fxy，C = fyy，Δ = AC - B²
	 Δ > 0 且 A > 0 → 极小值
	 Δ > 0 且 A < 0 → 极大值
	 Δ < 0 → 无极值

## 深度学习
* 深度学习定义：是机器学习的一个子领域，其核心思想是利用**多层神经网络**来模拟人脑处理数据的方式，从而自动从大量数据中提取特征并进行学习。
	之所以称为“深度”，是因为它使用的神经网络通常包含很多个隐藏层，这种深层结构使得模型能够学习从简单到复杂的层级化特征
	（例如：在图像识别中，底层识别边缘，中层识别形状，高层识别物体部件乃至完整物体）。

**基础神经网络的构成**
1. 输入层
- 作用：是将数据传递给下一层。
- 构成：由若干个神经元（节点）组成，神经元的数量通常等于输入数据的特征维度（每一个样本（数据点）需要多少个具体的数值（指标））

2. 隐藏层
- 功能：负责核心的计算和特征提取。数据在这里经过加权求和、加上偏置，并通过激活函数进行非线性变换。
- 构成：
	- 深度：一个或多个**隐藏层**。如果有很多层，就是“深度”网络
	- 神经元：每一层包含若干个**神经元**
	- 连接：上一层的每一个神经元都与下一层的每一个神经元相连（全连接），每条连接上都有一个权重 (Weight)，代表连接的强弱。每个神经元还有一个偏置 (Bias)。
- 关键操作：
	1. 线性变换： 𝑧=∑(输入×权重)+偏置
	2. 非线性激活： 𝑎=激活函数(𝑧)。常用的激活函数有 ReLU, Sigmoid, Tanh 等。激活函数是神经网络能拟合复杂曲线的关键，如果没有它，无论多少层网络都等价于单层线性模型。
- 作用：
	- 第一层隐藏层可能学习到简单的特征（如线条、颜色边界）。
	- 第二层可能组合这些简单特征形成形状（如圆形、方形）。
	- 更深层则组合成高级语义概念（如眼睛、车轮）。

3. 输出层
- 功能：产生最终的预测结果
- 构成：神经元的数量取决于任务类型。
	- 二分类问题（是/否）：通常只有 1 个神经元，输出一个 0 到 1 之间的概率值
	- 多分类问题（猫/狗/鸟）：神经元数量等于类别的数量。例如分3类，就有3个神经元，输出每个类别的概率分布
	- 回归问题（预测房价）：通常 1 个神经元，输出具体的数值

## 数据集的划分
- 通常不会把所有数据都用来训练模型，而是将数据划分为三个独立的子集：训练集（Training Set）、验证集（Validation Set）和测试集（Test Set）
- 核心目的：模拟模型在未知数据上的表现，防止模型“死记硬背”（过拟合），并确保评估结果的公正性。

**三大集合**
1. 训练集
 - 直接训练模型的数据子集，用损失函数调整自身
 - 占比：60%~80%
2. 验证集
 - 用于调整参数和监控训练过程，注：==**模型不会直接看到这部分的数据的标签来更新权重
3. 测试集
 - 用于最终评估模型性能的数据子集，使用一次来评估模型的泛化能力

如果不进行划分就会出现严重的**过拟合**（在训练集上表现完美，但在没见过的新数据上表现很差）的问题

## 神经网络结构

**基本组成结构：人工神经元**

人工神经元包含以下部分：
输入（Xn）、权重（Wn）、偏置（b）、激活函数（f）

e.g:
- 输入：朋友邀请你去聚会（X1），天气很好（X2），你有空（X3）
- 权重：你有多在乎这个朋友（W1）、你多喜欢好天气（W2）、空闲对你的重要性（W3）
- 偏置：你今天原本就不想动（ 偏置𝑏 是负数，增加出门难度）。
- 加权求和：计算总得分。
- 激活函数：如果总分超过某个门槛（比如60分），你就决定“去”（输出1）；否则决定“不去”（输出0或接近0的值）

神经元是基石、架构是骨架，通过将神经元分层并定义连接方式（全连接、卷积、注意力等），构成模型

## 前向传播
- 前向传播就是数据从输入端流入，经过层层计算和变换，最终从输出端流出结果的过程，网络的**参数**（权重和偏置）是**固定不变**的（无学习发生）

### 常见的激活函数
 没有激活函数就无法拟合曲线或复杂的决策边界
1. ReLU (Rectified Linear Unit) - 修正线性单元
 目前的首选
 公式：f(x)=max(0,x)
 优点：
 - 计算极快：只需要判断正负，没有复杂的指数运算。
 - 缓解梯度消失：在正区间梯度恒为 1，反向传播时误差能顺畅传回深层
 - 稀疏性：负值被置零，意味着部分神经元不工作，这类似于生物大脑的稀疏激活，有助于提取特征。
 适用于几乎所有隐藏层
2. Sigmoid (Logistic Function) - S型函数
 主用于输出层
 图像为平滑的“S”曲线，输出范围在（0,1） 
 公式：
 $$
 f(x)=1/(1+1/e^x)
 $$
 缺点：
 - 梯度消失严重：当x很大活很小时，曲线平缓，**==梯度趋于0。导致误差传回签名几层时几乎消失，无法训练==**——==梯度消失🚫==
 - 输出不以0为中心，优化效率低
 - 计算较慢
 - 适用场景：二分类问题的输出层（输出预测为“是”的概率）。不推荐用于隐藏层。
3. Leaky ReLU & Parametric ReLU (PReLU)
 为了解决ReLU“神经元死亡”问题而设计的变体

 解决了 Dead ReLU 问题，即使输入为负，梯度也不为 0，神经元依然可以学习

|:--|:--|:--|:--|
|隐藏层|通用/CNN/MLP|ReLU|计算快，收敛快，效果好。首选。|

|输出层|多分类 (A/B/C...)|Softmax|输出多个类别的概率分布，和为1。|
|输出层|回归 (预测数值)|Linear (无激活)|直接输出数值，不限制范围。|
## 反向传播
反向传播是神经网络训练的**核心算法**

- 网络刚初始化时，权重是随机的。通过==**反向传播 + 梯度下降**==让调整权重，让预测值无限接近真实值。

### 工作流程
1. 前向传播一次，将预测值（Predicted 𝑦^​）与真实标签（True Label 𝑦）进行比较
2. 使用**损失函数**量化误差，预测越离谱𝐿𝑜𝑠𝑠越大。常见公式（均方误差）：𝐿𝑜𝑠𝑠=（𝑦^−𝑦)^2
3. **倒着去追究责任**：
	链式法则（Chain Rule）：利用微积分中的链式法则，从输出层开始，一层层往回推算
	计算梯度（Gradient）：
	- 计算损失函数对每个权重的偏导数
	- 计算梯度来决定增加还是减少以及调整幅度大小
	如果输出层的某个神经元错误很大，反向传播会将大部分“锅”甩给连接到它的上一层权重。
4. 参数更新
- 使用**优化器**来更新权重

一次“前向+反向+更新”只能处理一个批次（**Batch**）的数据
**epoch**（轮次）：当整个训练集的所有数据都被这样“练”了一遍，称为一个 Epoch
停止条件：当 Loss 不再下降，或者**验证集效果**开始变差（过拟合），停止训练。
## （）监督学习

- 监督学习：像有老师辅导的学生。老师给题目（输入）和标准答案（标签），学生做题，做错了老师纠正，直到学会。
- 无监督学习：像自学探索的孩子。只有一堆玩具（数据），没有说明书。孩子自己观察玩具的颜色、形状，把它们分类或找出规律。
- 半监督学习：像只有少量例题的学生。老师给了几道带答案的题，但大部分是只有题目没答案的习题。学生利用那几道例题的规律，去推导剩下大量习题的结构。
### 监督学习
定义：使用带有标签的数据集进行训练。模型学习输入 𝑋 （特征）到输出 𝑌（标签）的映射关系。
目标：预测新数据的标签或数值
优缺点：
- ✅ 准确率高，目标明确，评估容易
- ❌ 依赖大量高质量标注数据，标注成本极高（需要人工打标签）
常见算法：线性回归、逻辑回归、支持向量机 (SVM)、决策树、随机森林、神经网络 (CNN, RNN, Transformer)。

### 无监督学习
定义：使用没有标签的数据集进行训练。模型自行发现数据内部的结构、模式或分布。只有 𝑋，没有 𝑌
目标：理解数据的内在结构，压缩数据，或生成新数据。
主要任务：
- 聚类：将相似的数据归为一组
- 降维：在保留主要信息的前提下减少特征数量
- 关联规则：发现数据项之间的联系
- 生成模型：学习数据分布并生成新样本
常见算法：K-Means, 层次聚类, PCA, 自编码器 (Autoencoder), GAN
优缺点：
- ✅ 不需要人工标注，数据获取容易，能发现人类未知的模式。
- ❌ 结果**难以量化评估**（没有标准答案），效果往往**不如监督学习精准**。

### 半监督学习
定义：结合少量带标签数据和大量无标签数据进行训练。旨在利用无标签数据来提升模型性能，同时==**降低标注成本**==。
目标：在标注数据稀缺的情况下，达到接近全监督学习的效果
伪标签 (Pseudo-labeling)：用已训练的模型给无标签数据打“伪标签”，置信度高的加入训练集迭代。


## 拓展内容
### 数据归一化
作用：
- 加速收敛：让损失函数等高线变圆，梯度下降能走直线直达最低点。
- 提高精度：防止大数值特征主导模型，让所有特征权重平等。
- 避免梯度消失：将数据控制在激活函数的敏感区间内。
- 统一量纲：让不同单位的数据（如“米”和“千克”）具有可比性。
深度学习几乎必须归一化

### 正则化
- 目的：在“模型对训练数据拟合得很好”和“模型结构简单”之间寻找平衡，从而让模型在面对从未见过的测试数据时，依然能表现良好（即**提高泛化能力**）。通过惩罚模型的复杂度，迫使模型**学习更通用的规律**

正则化之前公式：MinimizeJ(θ) = L(θ)
正则化之后公式：MinimizeJ(θ) = L(θ) + λ * R(θ)
- L(θ)——误差项
- R(θ) ：正则化项。衡量模型的复杂度
- λ (Lambda)：正则化系数。控制惩罚的力度

两个主流正则化方法对比

| **特性**   | L1 正则化 (Lasso)    | L2 正则化 (Ridge)     |
| -------- | ----------------- | ------------------ |
| **解的特性** | **稀疏解** (许多权重变为0) | **稠密解** (权重很小但不为0) |
| **几何形状** | 菱形约束区 (角点易接触坐标轴)  | 圆形约束区 (平滑接触)       |
| **主要功能** | **特征选择** + 防过拟合   | **防止过拟合** + 权重平滑   |
| **可导性**  | 在0处不可导 (需特殊优化算法)  | 处处可导 (梯度下降友好)      |
| **常用场景** | 特征筛选、高维稀疏数据       | 深度学习、通用回归分类        |

### Batch Size，学习率
#### Batch Size（批次大小）
通常从 32 或 64 开始尝试。如果显存允许，可以适当增大以加速训练，但过大的 Batch Size 往往会导致模型泛化性能下降（Generalization Gap）。

#### Learning Rate（学习率，LR）
定义：控制模型参数在每次梯度更新时变化幅度的系数。
公式：
𝜃𝑛𝑒𝑤=𝜃𝑜𝑙𝑑−𝜂⋅∇𝐽(𝜃)θnew​ = θ old​ −η⋅∇J(θ)

学习率策略：
固定学习率往往不是最优的，现代训练通常使用动态调整策略：
- 热身 (Warmup)：刚开始训练时，用很小的 LR，慢慢增大。防止初始梯度不稳定导致模型崩溃（Transformer 模型标配）。
- 衰减 (Decay)：
	- Step Decay：每隔几个 Epoch，将 LR 除以 10（如 0.1 -> 0.01 -> 0.001）。
	- Cosine Annealing：让 LR 按余弦曲线从大到小平滑下降。
	- ReduceLROnPlateau：监控验证集 Loss，如果不再下降，自动减小 LR。

#### 两者的相互作用
如果你将 Batch Size 扩大k倍，为了保持梯度的期望方差一致，通常也需要将 Learning Rate 扩大𝑘倍。
因为大 Batch 算出的梯度更准（噪声小），所以可以大胆地迈大步（大 LR）；小 Batch 噪声大，必须小步慢走（小 LR）以防跑偏。
所以在训练中先定Barch Size再调LR
