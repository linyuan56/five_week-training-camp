### 方法解析顺序（MRO）：当类有多个父类时，python使用C3线性化算法确定方法调用的顺序
e.g.
```
	class A:
		def method(self)
	classB(A):
		def method(self)
		# 调用父类A的方法
	classC(A):
		def method(self)
		# 调用父类A的方法
	class D(B,C):
		def method(self)
		# 调用父类的方法
	
	d = D()
	d.method() # 调用顺序是 D->B->C->A
```
### 继承的高级应用
1. 抽象基类（ABC）
	python提供abc模块，允许定义抽象基类，强势子类实现特定的方法，不实现就报错
e.g.
```
from abc import ABC, abstractmethod

class Shape(ABC):
	@abstractmethod
	def area(self):
		"""计算⾯积的⽅法"""
		pass
	
	@abstractmethod
	def perimeter(self):
		"""计算周⻓的⽅法"""
		pass

class Rectangle(Shape):
	def __init__(self, width, height):
		self.width = width
		self.height = height

	def area(self):
		return self.width * self.height
	def perimeter(self):
		return 2 * (self.width + self.height)
```
2. 组合vs继承
	组合表示“有一个”关系，一个类包含另一个类的实力作为其属性
	继承表示“是一个“关系，子类继承父类的所有公共属性和方法
```
# 错误：汽车“继承”引擎？语义崩坏！
class Car(Engine):  # Car 是 Engine？❌
    pass

# 正确：汽车“包含”引擎
class Car:
    def __init__(self):
        self.engine = Engine()
```

| 场景       | 选继承                                     | 选组合                            |

### 读取CSV文件
csv文件为常见的数据存储格式，用read_csv()函数来读取

```python
import pandas as pd

# 从 CSV 文件中读取数据
df = pd.read_csv('example.csv')

# 查看前几行数据
print(df.head())
```
一般用df.['A']来指代列，用df.loc[0]（索引为0的那一行）和df.iloc[0]（物理上的第一行）来指代行
#### read_csv 的不同参数：
- **`sep` 或 `delimiter`**：用以指定分隔符，默认逗号。sep = '\t'
- **`header`**：指定哪一行作为列名，默认为0（文件第一行），如果文件没有列名，可以设置header = None，然后通过names参数传入列名列表
- **`index_col`**：指定哪一列作为索引列。index_col = 0就是将第一列作为索引
- **`usecols`**：指定只读取文件中的部分列。usecols = [0，2，3]就是只读取第一三四列
- **`dtye`**：指定列的数据类型，强制某列为指定类。如{‘列名’：np.float64, '列名2'：np.int32}
- **`parse_dates`**：尝试将指定列为日期类型
- **`na_values`**：指定那些值被视为缺失值。navalues = ['NA', 'N/A', 'null']将括号内的视为缺失值
最后的形式：
```python
df = pd.read_csv('no_header.csv', header=None, names=['姓名', '年龄', '城市'])
```
这将读取 `no_header.csv` 文件，不将第一行作为列名，而是使用我们提供的列表 `['姓名', '年龄', '城市']` 作为列名。
- 读取excel，JSON，HTML表格，SQL数据库也是基本同理
### 数据清洗
1. 缺失值处理
定义：NAN（Not A Number）

通过`isnull()` 和 `notnull()` 方法来**检测缺失值**。会返回布尔类型的DataFrame（数据库）

处理缺失值的方法主要有两类：**删除**和**填充**。
1.1 **删除操作**：
通过`dropna()` 方法删除包含缺失值的行或列，参数如下：
- `axis`：指定删除行还是列。默认为 `0`，表示删除行；设置为 `1` 则删除列。
- `how`：指定删除条件。默认为 `'any'`，表示只要有任何缺失值就删除整行或整列；设置为 `'all'` 则表示只有当整行或整列全部为缺失值时才删除。
- `subset`：指定只考虑某些列的缺失值。例如，`subset=['A']` 表示只根据列 `A` 是否有缺失值来决定是否删除行。如subset = ['年龄']，若该列的一个值为NaN，则将这个值所在的行删除
- `inplace`：布尔值，默认为 `False`。如果设置为 `True`，将直接在原 DataFrame 上进行删除操作。
最后的形式：
```python
df_cleaned = df.dropna()
```
1.2 **填充操作**：
通过`fillna()` 方法可以将缺失值**替换**为指定的值或统计量，策略如下：
- 使用固定值填充：例如 `df.fillna(0)` 将所有缺失值替换为0。
- 使用均值、中位数或众数填充：例如 `df.fillna(df.mean())` 将缺失值替换为该列的均值。
- 使用前一个非缺失值或后一个非缺失值填充：可以通过 `method='ffill'`（前向填充）或 `method='bfill'`（后向填充）实现
最后形式：
```python
df_filled = df.fillna(df.mean())
```
2. 重复值处理
定义：数据集中完全形同的记录

通过`duplicated()` 方法来标记重复行，默认情况下，它会将第一次出现的重复行标记为 `False`，后续出现的标记为 `True`

通过`drop_duplicates()` 方法可以删除重复行，只保留第一次出现的行，参数如下：
- `subset`：指定只考虑某些列来判断重复。
- `keep`：指定保留哪个重复行。默认为 `'first'`，保留第一次出现的行；设置为 `'last'` 则保留最后一次出现的行；设置为 `False` 则删除所有重复行。
- `inplace`：布尔值，是否直接在原 DataFrame 上进行操作。
最后形式：
```python
df = pd.DataFrame(data)
df_keep_false = df.copy()
result_false = df_keep_false.drop_duplicates(subset=['姓名', '城市'], keep=False)
```
3. 数据类型转换与格式统一
3.1 **转换**操作：
一般通过`astype()` 方法可以将列转换为指定的数据类型。如astype(int)就是转换为数据类型，但如果有值没法转换，则会报错

在面对转换为数值类型的情况，就使用`pd.to_numeric()` 函数（更为灵活，但只能转化为数值类型），并设置`errors='coerce'` 参数，将无法转换的值设为 `NaN`，不会报错

3.2 **统一**操作：
- **日期格式统一**：可以使用 `pd.to_datetime()` 将字符串转换为日期类型
- **字符串格式统一**：对于字符串列，可以使用字符串方法来处理。例如，`str.strip()` 可以去除前导和后导空格；`str.lower()` 可以将字符串转换为小写，以统一大小写。
